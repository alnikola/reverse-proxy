Class RateLimitingQueueBase<TItem>

Class RateLimitingQueueBase is a delegating base class for IRateLimitingQueue<TItem> interface. These classes are ported from go, which favors composition over inheritance, so the pattern is followed. Implements the DelayingQueueBase<TItem>. Implements the IRateLimitingQueue<TItem>.

Forget(TItem)

Forget indicates that an item is finished being retried. Doesn't matter whether it's for perm failing or for success, we'll stop the rate limiter from tracking it. This only clears the rateLimiter, you still have to call Done on the queue.
